# DevShowcase Studio

DevShowcase Studio is a complete GitHub portfolio builder and developer productivity dashboard built using HTML, CSS, and JavaScript.

This project helps developers manage portfolio projects, track technical skills, organize roadmap tasks, and generate a professional GitHub profile README. It is designed for developers who want to improve their GitHub profile, organize their work better, and present their projects in a more professional way.

## Project Overview

DevShowcase Studio is a frontend web application that allows users to manage their GitHub portfolio in one place.

The app includes a project showcase manager, skill progress tracker, roadmap task board, and README generator. It uses browser LocalStorage to save user data, so users can continue managing their portfolio without needing a backend or database.

This project is suitable for:

- Web developers
- Students
- GitHub users
- Portfolio builders
- Frontend developers
- Beginner to intermediate JavaScript learners
- Developers who want to improve their GitHub profile

---

## Main Purpose

The purpose of DevShowcase Studio is to help developers organize their GitHub portfolio and improve their profile presentation.

Many developers create projects but do not properly document or organize them. This tool helps users manage project details, track progress, list skills, plan improvements, and generate a professional README profile.

## How the App Works

DevShowcase Studio works fully in the browser.

Users can add their portfolio projects, skills, and roadmap tasks. All data is stored locally using the browser's LocalStorage feature.

The app does not require login, server setup, database, or external API.

## App Sections

### 1. Dashboard

The dashboard shows a quick overview of the user's portfolio progress.

It displays:

- Total projects
- Completed projects
- Average project progress
- Open roadmap tasks
- Today mission
- Portfolio profile summary

---

### 2. Project Showcase Manager

The project manager allows users to add and organize GitHub projects.

Each project includes:

- Project name
- Category
- Status
- Progress percentage
- Description

### 4. Developer Roadmap Board

The roadmap board helps users plan GitHub and portfolio improvements.

## How to Run Locally

No installation is required.

Follow these steps:

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Start using the dashboard.

---

## How to Deploy on GitHub Pages

Follow these steps to deploy the project:

1. Create a new GitHub repository named `devshowcase-studio`.
2. Upload these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Commit the files to the `main` branch.
4. Go to repository `Settings`.
5. Open `Pages`.
6. Under `Build and deployment`, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Click `Save`.
8. Wait for GitHub Pages to publish the website.
9. Copy the published link and add it to your repository website field.
